Seven.io Notifier
=================

Provides [Seven.io](https://www.seven.io/) integration for Symfony Notifier.

DSN example
-----------

```
SEVENIO_DSN=sevenio://API_KEY@default?from=FROM
```

where:
 - `API_KEY` is your seven.io API key
 - `FROM` is your sender (optional, default: SMS)
